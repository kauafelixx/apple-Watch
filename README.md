# apple-Watch
Apple Watch - Página de Produto Este projeto é uma recriação da página de produto do Apple Watch, feita em HTML, CSS e JavaScript puro, com foco em interatividade e fidelidade visual.  O usuário pode:  Selecionar diferentes cores do produto (imagens e nome mudam dinamicamente).  Escolher o tamanho da caixa (41 mm ou 45 mm) com animação de escala.  Alternar entre diferentes ângulos/imagens do relógio.  Visualizar preço, parcelas, entrega e opções adicionais, tudo em um layout inspirado no site oficial da Apple.  Tecnologias utilizadas:  HTML5  CSS3  JavaScript (DOM e manipulação dinâmica de conteúdo)  Objetivo: Praticar manipulação do DOM, estilos dinâmicos e estruturação semântica de um layout profissional.
